# Credit Card Fraud Detection using Deep Learning

## 📌 Project Overview

This project is a Credit Card Fraud Detection System developed using Deep Learning techniques to identify fraudulent transactions accurately. The model analyzes transaction patterns and predicts whether a transaction is legitimate or fraudulent in real time.

The system is deployed with an interactive **Streamlit Web Application**, allowing users to test transactions easily through a modern and user-friendly interface.

---

## 🚀 Features

* Deep Learning based fraud detection
* Real-time transaction prediction
* Streamlit web interface
* Manual transaction input mode
* Demo mode using dataset samples
* Fraud probability score display
* Fast and accurate predictions

---

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Matplotlib / Seaborn

---

## 📂 Project Files

* `app.py` → Main Streamlit application
* `model.h5` → Trained Deep Learning model
* `scaler.pkl` → Data preprocessing scaler
* `creditcard.csv` → Dataset file
* `requirements.txt` → Required libraries

---

## ▶️ How to Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
